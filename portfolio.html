PRODUCTION LINE VISUALIZATION
My first introduction to 3D was at ThinkDigital when I was asked to develop a visual representation of one of our clients production lines.

Using real photos as a reference, I created low poly versions of each machine in several production lines. The renders would later receive color overlays to signal any machine malfunction.
▪ Renders made in Cinema4D and later 3ds Max
▪ Low poly machines modeled from scratch
▪ Balance between visual fidelity and clear understanding
▪ Hand traced SVG overlays for each machine to display alert colors
